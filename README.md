# Gym-LACOBRA

## Descripción

Gym-LACOBRA es una aplicación web desarrollada como proyecto académico para la gestión de información de un gimnasio.

El sistema utiliza una arquitectura cliente-servidor, con un frontend desarrollado en React y un backend desarrollado en Django, permitiendo la administración de datos mediante una interfaz web moderna.

## Instalación

### Backend

```bash
cd Backend
pip install -r requirements.txt
python manage.py runserver
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```
